## Relio CRM is live

Relio is a CRM purpose-built for commercial real estate teams. Here's what shipped in the initial release.

### New features

- **Interactive map view** — Plot properties on a Mapbox-powered map, draw custom territory boundaries, and color-code pins by pipeline stage, property type, or tags. Click any parcel to see the property record, owner, and deal status. [Learn more](/quickstart)

- **Relationship intelligence** — Every contact has a connection strength score (0–100) calculated from calls, emails, meetings, and linked records. See a contact's full network of properties, companies, and deals in one place.

- **Deal pipelines** — Track deals through custom stages designed for CRE workflows. Kanban and table views, deal analytics with pipeline value and win-rate metrics, and linked properties and contacts on every deal card.

- **Ask Relio (AI assistant)** — Chat with an AI assistant that can search your workspace, create and update records, run analytics queries, and perform web research including people search, company lookup, and SEC filings.

- **Real-time collaboration** — See who's online, who's viewing the same record, and who's editing a field. Colored cell borders, typing indicators, and field locking keep your team in sync.

- **Task management** — Create tasks with priorities, due dates, and assignees. View tasks in table, kanban, calendar, or Gantt chart layouts.

- **Custom lists** — Build static or smart lists for any record type. Pipeline-type lists support kanban views with configurable stages.

- **Agent sites** — Publish a public-facing property showcase on a custom subdomain or your own domain. Includes property listings, a contact form, and newsletter subscriptions.

- **Mobile app** — Access your pipeline, contacts, properties, and tasks from iOS and Android with a native app built on React Native.

- **Data import and export** — Import contacts, properties, companies, and deals from CSV with column mapping and duplicate detection. Export workspace data in bulk.

### Integrations

- **Regrid** — Connect your Regrid API key for nationwide parcel data, owner lookup, and parcel boundary overlays on the map.

- **Smarty** — Address verification and autocomplete powered by Smarty.

- **Developer API and webhooks** — Generate API tokens with granular permissions and subscribe to events via webhooks. [Learn more](/api-reference/introduction)
